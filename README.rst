Tarkus
======

Tarkus is a plugin manager for IDA Pro, modelled after Python's pip.

Getting Started
---------------

Install...

.. code:: bash
    
    pip install...
    
Initialize it:

.. code:: bash

    tarkus init "C:\Program Files (x86)\IDA 6.8"
    
And start installing plugins:

.. code:: bash

    tarkus install autostruct
    
That's it!
